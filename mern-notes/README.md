# NoteForm Uygulaması 📝

Bu proje, React kullanarak bir not ekleme formu oluşturur.  
Kullanıcı, not başlığı ve açıklamasını yazarak sisteme ekleyebilir.  

## 🚀 Kullanılan Teknolojiler  
- React.js  
- Fetch API
